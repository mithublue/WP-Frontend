Frontend profile builder, authentication and post submission plugin for wordpress. Evenrything is in frontend.

== Description ==

Want to let users create post from and have their own dashboard in frontend and prevent them accessing the admin panel ? Are you bored in default wp login and registration and want
custom ones to match with your theme ? These are exactly what this plugin does.

Additionally, you will have lots of options to customize everything to your needs.

== Demo ==

You can try the full featured demo of this plugin <a href="http://cybercraftit.com/demo/testwp/wp-frontend-pro/demo/">here</a> .
There you will find 'WP Frontend' in admin menu from where you can create post form and use that in any post or page or any other post type. You will also have different settings there. Playing with those will let you be clear about the plugin.

== WP Frontend Features ==

=== User can have their own dashboard in front ===

Registered and logged in users will have their custom dashboard in frontend where they can see their posts, edit them, delete them and even create new post.
You can control their actions from backend.

=== User can create post from frontend ===

User can create post from frontend. ANY post of any POST TYPE and even PAGE can be creted from frontend by user. Admin will have option to decide if
post creation will require login or not.

=== Create post as guest ===

There is option to let user create post as guest. Admin also has option to chose default post author for the post created by the guest post.

=== User can be restricted on accessing the admin panel ===

Admin can chose roles whom he wants to give access to admin panel whom he does not.

=== Unlimited forms ===

Form is needed to create post. User will use that form to create any post. With WP Frontend, admin can create form for any post type. Admin can create as many forms as he wants.

=== You can build form to create post of any post type ===

Admin can create form to create any post of any post type along with product, page, post etc. For example: If admin wants user
to create page, he will craete page form for them.  User will create page from frontend by the form created in admin panel.

=== Numerous fields ===

Lots of fields are there in the form builder to create form for post. 30+ Fields ( 16 fields in free version ) currently are supported.

=== Drag and drop. No coding required ===

No coding , no worry, drag and drop form builder for you.

=== Visual form builder. You will see the output in realtime ===

=== Blazing fast admin panel ===

Page reloading is boring and time consuming, so is to us. That's why we have kept focus on smooth and fastest user experience, and for this purpose, WP Frontend
is made as non reloading app.

=== Ajax form submission. Form can be submitted without page refresh ===

=== Schedule form submission ===

Admin can bind post creation in time limit. He can set the schedule within which the form will be accessible

=== Set redirection after successful post submission ===

=== Customizable message on post submission ===

=== Limit post submission ===

Admin can set number of post submission/creation using a specific form.

=== Suitable to any theme ===

=== Responsive ===

=== 24 grids support ===
